## 🎯 项目概述

一个基于 Flask + WebSocket 的**实时翻译可视化平台**，展示 AI 驱动的长文本智能分块翻译。

**核心价值**: 将传统的批处理翻译转变为可视化、实时、智能的交互体验。

---

## ✨ 核心功能 (8大特性)

### 1. 🧠 智能分块系统
- **三级 Fallback 策略**:
  - Level 1: 二级标题 (##) 分割 ⭐ 首选
  - Level 2: 一级标题 (#) 分割 🔄 备选
  - Level 3: 固定大小 (8000 chars) ⚠️ 兜底
- **智能边界检测**: 不截断句子和段落
- **章节信息保留**: 每个 chunk 携带章节元数据

### 2. 🌍 24 种语言支持
- **按地区分组**:
  - 🌏 东亚 (4): 中文简繁、日韩
  - 🇪🇺 欧洲 (9): 法德西意葡俄等
  - 🕌 中东南亚 (6): 阿拉伯、印地、波斯等
  - 🌴 东南亚 (4): 泰越印马
- **全球覆盖**: 49 亿人口 (63%)
- **多文字系统**: 拉丁、阿拉伯、希伯来、CJK、天城、泰文

### 3. 🔄 混合术语模式
- **精选术语库**: 90 个人工精选术语
  - 30 专有名词
  - 46 技术术语
  - 14 关键概念
- **动态提取**: 从首块自动发现新术语
- **智能合并**: 过滤重复，最终 ~96-105 个术语
- **实时日志**: 清晰展示提取过程

### 4. 📊 实时可视化
- **三栏布局**:
  - 左: 上传区 + 配置 + 分块列表
  - 中: 实时日志流（彩色、可滚动）
  - 右: 进度条 + 统计信息 + 下载
- **WebSocket 推送**: 零延迟实时更新
- **双层进度条**:
  - 整体进度 (1/30 chunks)
  - 当前块进度 (0-100%)

### 5. 🎨 现代化 UI
- **黑暗科技主题**: Slate 灰 + 霓虹蓝绿
- **Tailwind CSS**: 响应式设计
- **动画效果**: 
  - Glow border (发光边框)
  - Gradient buttons (渐变按钮)
  - Smooth transitions (平滑过渡)
- **拖拽上传**: Drag & Drop 支持

### 6. ⚡ 性能优化
- **流式处理**: 每块完成立即保存
- **增量保存**: 支持断点续传
- **上下文传递**: 最后 2 段作为连接
- **重叠检测**: 避免重复翻译
- **异步处理**: 后台线程 + WebSocket

### 7. 🛡️ 容错机制
- **文件大小限制**: 50MB
- **超时控制**: 60 分钟
- **错误日志**: 彩色标记（红色）
- **状态追踪**: idle/analyzing/translating/completed/error
- **任务恢复**: 保存中间结果

### 8. 📥 便捷下载
- **自动命名**: `原文件名_语言.md`
- **一键下载**: 绿色按钮
- **格式保留**: Markdown 结构完整

## 🚀 部署就绪

### 本地部署
```bash
cd Master-Translator-Web
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
# 访问 http://localhost:5001
```

### Docker 部署 (可选)
```dockerfile
FROM python:3.13-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
EXPOSE 5001
CMD ["python", "app.py"]
```
