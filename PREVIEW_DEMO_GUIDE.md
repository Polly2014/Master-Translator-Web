# 🎬 预览功能快速演示指南

## 快速开始 (5分钟演示)

### 1️⃣ 启动应用
```bash
cd Master-Translator-Web
python app.py
```

访问: http://localhost:5001

### 2️⃣ 准备测试文件
使用我们的快速演示文件（60-90秒翻译）：
```
demo_files/Mustafa_Book_Quick_Demo.md
```

### 3️⃣ 执行翻译
1. 📤 **上传文件**: 拖放或点击上传区域
2. 🌐 **选择语言**: 默认 Japanese
3. 🔍 **分析文档**: 点击 "Analyze Chunks"
4. 🚀 **开始翻译**: 点击 "Start Translation"
5. ⏳ **等待完成**: 观察进度条到达 100%

### 4️⃣ 测试预览功能 ⭐

#### 看到下载面板
```
┌──────────────────────────────┐
│ 👁️  Preview Translation      │ ← 新功能！
├──────────────────────────────┤
│ 📥  Download Translation     │
└──────────────────────────────┘
```

#### 点击 "Preview Translation"
预览窗口会立即打开，展示翻译结果！

### 5️⃣ 体验功能特性

#### ✨ 切换视图
- **Rendered** (默认): 查看格式化的内容
  - 标题层级分明
  - 段落自动换行
  - 列表清晰排列
  
- **Raw Text**: 查看 Markdown 原始代码
  - 等宽字体显示
  - 保留所有语法标记
  - 方便检查格式

#### 📊 查看统计
在窗口底部看到：
```
📊 1,234 words • 5,678 characters
```

#### 💾 下载或关闭
- **Download** 按钮: 预览窗口内直接下载
- **X** 按钮: 关闭预览
- **ESC** 键: 快速关闭
- **点击背景**: 也可关闭
